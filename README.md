# Scala and sbt Dockerfile

This repository contains the **Dockerfile** of
[iptables](https://netfilter.org/projects/iptables/index.html) on top of OpenJDK.


## Base Docker Image ##

* [openjdk](https://hub.docker.com/_/openjdk) (slim)
